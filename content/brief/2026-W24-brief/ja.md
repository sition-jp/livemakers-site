## エグゼクティブ・サマリー

W24 は、**監査ゲートの週**として読むのが最も正確である。市場は W23 の激しい暗号資産売りから反発したが、Cardano にとって最重要の進展は価格ではなくプロセスだった。2026 年予算の投票フェーズは Hydra Voting 上で完了し、**5B ADA 超の投票力、active DRep stake の約 85%、100 名超の DRep 参加**を記録した。これは Voltaire にとって大きな正統性のシグナルである。ただし、これは最終的な予算結果ではない。正しい次の段階は **2026-06-15 から 2026-06-19** に予定される独立監査であり、その後に統合とオンチェーン提出準備が続く。

相場は改善したが、きれいな risk-on には戻っていない。W23 基準スナップショットから W24 までに **BTC は +3.5%、ETH は +5.1%、ADA は +6.6%** 上昇し、W24 取得時点の ADA 24h 変化率は +6.84% だった。一方で S&P 500 は前週比 -2.5%、VIX は 15.4 から 19.44 へ上昇し、USDJPY は 160 近辺に残り、Iran については「合意」見出しから条件検証へ移ったことで原油は反落した。暗号資産は反発したが、マクロリスクは生きている。

価格の裏側で、Cardano の実行レイヤーは前進を続けた。Cardano Vision 2026 に関する IOR 提案は **74.96% の confidence vote** で批准された。Essential Cardano の 2026-06-12 開発レポートは、High Assurance による property-based testing tool の beta、Mithril の **SNARK-friendly genesis certificate**、DMQ support、Cardano-node ledger-state certification の進捗を示した。Constitutional Committee 選挙も注視点となった。9 月に 4 議席が空席となる一方、レポート時点の候補者登録は 4 名のみで、候補者締切は **2026-06-21** である。

規制は建設的だが未完である。Lummis 上院議員は CLARITY を、既に存在するルールを法律へ変える作業として位置づけた。日本の金融庁は銀行法施行規則等の一部改正に関する公布とパブリックコメント結果を公表した。これらはレールであって、単独の価格カタリストではない。Midnight は、W23 の開発者・エージェント層からコミュニティ拡張へ進み、日本の Nightforce Cohort 5 募集が始まった。ただし NIGHT は 24h では反発しつつも、W23 基準価格との比較では前週比で低い。正直な読みは従来どおりだ。注目は存在するが、利用の証明が転換テストである。

したがって W24 の結論は、「risk-on が戻った」というより狭く、より実用的である。**市場は W23 のパニックを止めるだけの反発を見せたが、投資の時計は検証へ移った。** 来週のハードゲートは二つ、予算監査ウィンドウと 2026-06-16/17 の FOMC である。Cardano では、高参加率のオフチェーン投票が、監査済みのオンチェーン提出パッケージへきれいに変換されるかを確認する週になる。市場では、VIX 20 近辺、USDJPY 160 近辺、未解決の地政学条件の中で、暗号資産の反発が持続するかを見る週になる。

### 反発は本物だが、リスクレジームはリセットされていない

W23 との違いは明白だ。暗号資産は一直線の下落を止めた。BTC、ETH、ADA、SOL、XRP はいずれも反発し、W24 取得時点では資産クラス全体に強い 24h 変化率が出ていた。ADA の前週比 +6.6% は、前週の -31% ドローダウン後だったため意味がある。少なくとも、W23 の売りが基準時点でさらに加速していたわけではない。

しかし、これをきれいな risk-on と呼ぶのは早い。株式の地合いは W23 の「最高値で平静」から悪化した。SPX -2.5%、Nasdaq 総合 -3.8%、DJI -1.4%、VIX は前週比 +26%。USDJPY は、W23 がすでにマクロ・トリガーとして挙げた 160 近辺に残った。原油は、Iran について市場を動かした合意見出しが条件検証と反対メッセージへ移ったことで大きく下げた。この組み合わせはより繊細なレジームを示す。**暗号資産は反発したが、マクロ背景は平静な乖離から未解決の検証へ移った。**

投資上の有用な区別は、**価格修復**と**レジーム修復**である。W24 が与えたのは価格修復であって、レジーム修復ではない。後者には、ボラティリティの持続的な低下、よりクリーンな USDJPY 環境、中東・原油条件の安定、そして来週の FOMC が流動性ストーリーを引き締めないことが必要になる。

## 2. ガバナンス・アップデート — 2026 予算投票は閉じたが、結果はまだ最終化されていない

### 見出しの数字は参加率である

W24 における Cardano 最大の出来事は、Hydra Voting 上で 2026 年予算の投票フェーズが閉じたことだった。Intersect は **5B ADA 超の投票力**、**active DRep stake の約 85%**、**100 名超の DRep 参加**を報告した。ローカル調査キューで引用された 2025 年予算サイクル比較も上回っており、Voltaire のオフチェーン調整メカニズムにとって正統性の節目である。

それがポジティブな読みである。同じくらい重要なのが、正確な読みだ。**投票完了は、予算結果の最終化を意味しない。** プロセスは次に、**2026-06-15 から 2026-06-19** に予定される独立監査へ進む。監査後に、統合とオンチェーン提出準備が必要になる。したがって正しい見出しは「次は監査」であり、「予算可決」や「予算否決」ではない。

この区別が重要なのは、Voltaire の信頼性が、高参加率のガバナンスを再現可能で監査可能な形にし、オンチェーンアクションへきれいに写像できるかにかかっているためだ。W23 は Summit 投票で「賛成多数」と「資金放出」が同じではないことを教えた。W24 は隣接する教訓を示す。**オフチェーン投票完了は、監査済みオンチェーン準備完了と同じではない。**

### IOR / Cardano Vision 2026 は批准された

研究ガバナンスでは、Cardano Vision 2026 に関する IOR 提案が **74.96% の confidence vote** で批准された。これにより Vision 2026 は、緩い戦略メモより強いマンデートを得た。より重要な読みは、Cardano の研究レイヤーが、他方で進む予算・実装・ガバナンスの議論と整合し始めていることだ。vision、funding、testing、governance はいずれも、非公式な拍手ではなく正式なチェックを通るようになっている。

### Constitutional Committee 候補者供給が次のボトルネック

Essential Cardano の 2026-06-12 レポートは、静かだが重要なガバナンス問題も示した。9 月に Constitutional Committee の 4 議席が空席となり、レポート時点で登録候補者は 4 名のみ、候補者締切は **2026-06-21** である。危機ではないが、ボトルネックではある。成熟したガバナンスシステムには、席を埋めるだけの候補者ではなく、競争的な候補者供給が必要だ。

DRep と SPO にとって、これは W25 の実務的な注視点になる。締切前に、より多くの適格候補者が現れるか。そして 9 月の刷新が、能力補填ではなく本物の選択になるか。

## 3. ビルド層 — High Assurance と Mithril が静かなクリティカルパスを進める

W24 には、W23 の CME / Indigo クラスターに相当する単一の消費者向け Cardano 製品見出しはなかった。その代わり、クリティカルパスは assurance と infrastructure の深い層へ移った。

### High Assurance: property-based testing が beta へ

High Assurance チームは、内部テスト用の property-based testing tool の beta 版を完成させた。レポートは、testing interface、positive / negative testing、threat models、coverage、VS Code extension、AI skill の実験に触れている。派手ではない。だが、公的インフラへ移ろうとするネットワークにとって、まさに派手ではないこの作業が重要である。

Property-based testing はプロトコル作業で特に重要だ。テストを「この例が通ったか」から「生成された広いケース空間で、この性質が保たれるか」へ移すからである。ガバナンスが重く、hard fork 可能な public chain にとって、この種の規律は任意の配管ではない。プロトコル変更が信頼を得る方法の一部である。

### Mithril: SNARK-friendly genesis と DMQ support

2026-06-12 レポートにおける Mithril の行も戦略的に重要だ。チームは **SNARK-friendly genesis certificate** の実装を完了し、SNARK recursive circuit のためのリファクタリング、prover input、SNARK proof verification、DMQ ledger peer support、DMQ message ID format enforcement、Cardano-node ledger-state certification prototype を進めた。

短く言えば、Mithril は Cardano state をより強い proof 保証で certify、compress、verify し、軽量クライアントや隣接システムが消費できる世界へ進み続けている。これは、より良い wallet、bridge、rollup 的統合、cross-domain infrastructure の前提になる。そして、出荷した週に価格を動かすことがほとんどないタイプの進捗でもある。

### アーキテクチャリスクは低位維持

W24 のビルドニュースは assurance-heavy であるため、アーキテクチャリスクは大きく低下するというより **LOW →** を維持する。方向は前進だが、証明は integration にある。注視点は、High Assurance beta が実務的な review workflow へ入り、Mithril の SNARK / DMQ 作業が production path へ届き、これらがロードマップ上の立派な名詞に留まらず運用リスクを下げるかどうかである。

## 4. マクロ / 規制 — CLARITY、日本 FSA、Iran 条件、160 円ライン

### CLARITY: 建設的なフレームだが、成立ではない

Lummis 上院議員の W24 メッセージングは、CLARITY を、開発者・投資家・市場の integrity を守りながら既存ルールを法律へ変える作業として位置づけた。これは市場構造のナラティブにとって建設的である。ただし CLARITY が成立したという意味ではない。有用な読みは、ワシントンの暗号資産議論が「ルールは必要か」から「どのルールを法律にし、どのように当局権限を分けるか」へ移り続けていることだ。

Cardano にとっての関連性は短期価格ではない。機関投資家にとっての読みやすさである。より明確な米国の境界は、規制商品、custody、開発者責任分析、上場判断を助ける。W24 はそのフレームに勢いを加えたが、ファイルを閉じたわけではない。

### 日本 FSA: 銀行規則改正が公式記録へ

2026-06-12、日本の金融庁は、銀行法施行規則を含む内閣府令改正の公布とパブリックコメント結果を公表した。これは W23 の日本規制議論と同じ広い post-enforcement 環境に位置づくが、単独の新しい暗号資産カタリストとして過大に読むべきではない。

日本の takeaway はより運用的である。金融レールは、公式のパブリックコメントと府令プロセスを通じて明確化され続けている。stablecoin、銀行アクセス、custody、card、決済仲介がすべて重要な市場では、ゆっくりした公式記録も採用面の一部である。

### Iran: 完了した deal ではなく条件検証

W24 の原油の動きは、Iran ナラティブの修正として読むのが最もよい。初期の合意見出しは市場を動かしたが、その後の物語は条件検証になった。何が書かれ、何が合意され、Strait / oil / shipping insurance の条件が変わるのか、当局者が同じ terms を確認するのか。WTI -7.4%、Brent -7.0% はエネルギーリスクが低く再評価されたことを示すが、地政学リスクが消えたことを示すわけではない。

だから W24 は、きれいな risk-on の週ではない。暗号資産が反発し、原油が下がり、VIX が前週比で高いまま、という組み合わせは共存できる。しかし、それは all-clear ではない。

### USDJPY 160 と FOMC が来週のマクロゲート

USDJPY は W23 からほぼ変わらず 160 近辺に留まった。これは W23 が示した心理的・政策的に敏感なラインと同じである。**2026-06-16/17** の FOMC は二つ目のゲートになる。Fed のメッセージが金利・ドル環境を引き締めるなら、暗号資産の反発はより厳しい流動性テストに向かう。軟らかく着地するなら、W24 の反発はレジーム修復へ進む余地を得る。

## 5. Midnight ウォッチ — 貢献者拡張。ただし利用証明がなお決める

Midnight の W24 ノートは、W23 の開発者・エージェント層より狭いが、それでも有用である。**日本で Nightforce Cohort 5 の募集が始まった。** これは重要だ。Midnight が必要としているのは、core protocol の進捗だけでなく、ambassador、educator、developer、local community operator、そして最終的な application team へ続く反復的な contributor funnel だからである。

NIGHT の価格はバランスの取れた読みを与える。W24 取得時点の 24h では +6.78% 反発したが、W23 基準との比較では約 -6.2% だった。これは失敗でも確認でもない。単に、W23 の主張がまだ正しいことを意味する。NIGHT には注目と独自のナラティブがある。しかし、利用証明はなお転換テストである。

W25 の注視点は変わらない。信頼できる City V2 またはオンチェーン利用データ、開発者活動の測定可能な成長、bridge または DUST capacity milestone、そして日本の contributor funnel がコミュニティ運動だけでなく可視的な output へ変わるかである。
